# web-macaujc-notes-99f5

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。页面内容以笔记、信息整理或参考用途为主，不针对特定网站或域名。仓库仅供存档与展示使用，不构成任何形式的建议或推荐。

## 目录说明

```
.
├── README.md          # 本说明文件
└── pages/             # 存放各独立 HTML 页面
    ├── page-001.html
    ├── page-002.html
    └── ...            # 更多页面按需添加
```

- `pages/` 目录下放置所有 HTML 文件，每个文件为一个独立页面。
- 页面命名保持简洁，便于索引与引用。

## 页面归档说明

- 每个 HTML 文件均为独立内容，不依赖外部资源或脚本（除非在文件内自包含）。
- 页面内容仅代表归档时的状态，可能不反映后续更新或变化。
- 所有页面均以静态 HTML 形式保存，可直接在浏览器中打开查看。

## 维护说明

- 本仓库不定期更新，新增页面或修订已有内容。
- 不保证所有页面长期有效或内容准确，请以原始来源为准。
- 如您发现页面内容存在明显错误或过期信息，欢迎通过 Issues 或 Pull Requests 提出修正建议。
- 本仓库不接受任何形式的广告、推广或外部链接插入。

## 使用方式

您可以直接克隆或下载本仓库，在本地打开 `pages/` 目录下的任意 HTML 文件即可查看。如需部署到 GitHub Pages 或其他静态托管服务，请将 `pages/` 目录设为根目录。
